About Discboxd

Discboxd is a music review forum web application where users can create accounts, post reviews, comment on reviews, and interact with music-related content.

NPM Packages Used

  • express - web server and routing
  • mongoose - MongoDB object data modeling
  • cors - cross-origin request handling
  • express-session - session management
  • connect-mongo - MongoDB-backed session storage
  • bcrypt - password hashing
  • dotenv - environment variable management
  • nodemon - development auto-restart tool

Third-Party Libraries / Assets

  • MongoDB - database system
  • Node.js - JavaScript runtime
  • Custom local image assets for album covers

Project Notes

This project follows an MVC-style structure with models, routes/controllers, database-backed CRUD operations, authentication, session handling, and validation.
